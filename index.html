Programming Language Suggester

This language suggester application was made for prospective coders who are interested in learning which language they should learn first. The app is designed to suggest three languages the user should learn based on survey questions including personal questions and technical history.

Survey Questions

  1. How would you rate your usability competency using your computer?
  2. Are you learning to code for personal use or professional opportunities?
  3. Do you have prior experience or knowledge with any languages?
  4. What do you intend to develop through learning to code?
  5. Do you prefer elegance or functionality ?

Feel free to type in answers in the boxes or select one from the select menu!
